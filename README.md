# My Daily Hub
My Daily Hub adalah sebuah aplikasi mobile berbasis Flutter yang dikembangkan oleh Jihan Nur Alfiah NRP: 15-2024-215 sebagai tugas Ujian Tengah Semester pada Mata Kuliah Pemrograman Mobile. Aplikasi ini berisi berbagai fitur sederhana yang fungsinya menampilkan pusat aktivitas harian, terdiri dari splash screen, dashboard, biodata, kontak, kalkulator sederhana, halaman cuaca, dan daftar berita.

### Biodata
![WhatsApp Image 2025-11-12 at 08 59 02_f5eddea8](https://github.com/user-attachments/assets/737a9199-2aab-4419-9395-8a2538445451)

Halaman Biodata menampilkan informasi pribadi dan akademik pengguna, lengkap dengan foto profil, nama, NIM, kelas, dan program studi. Selain itu, pengguna juga dapat mengisi informasi tambahan seperti jenis kelamin, hobi/minat, dan tanggal lahir menggunakan elemen interaktif radio button, dropdown, text field, dan kalender. Data bersifat statis, namun desainnya sudah siap dikembangkan menjadi halaman input dinamis di masa depan.
